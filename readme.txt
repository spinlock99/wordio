Wordio is a simple plugin that lets anyone send you a text message from you blog.

== Description ==

Wordio is a simple plugin that gives you a shortcode which you can place on any page. That
page will then present visitors with a form that lets them send you a text message.

== Installation ==

1. Install the Plugin from the WordPress Plugin Directory.
2. Activate the Plugin.
3. Navigate to the Twilio Settings page in the Admin page (Settings > Twilio Settings).
4. Enter your Twilio Account information.
5. Place `[wordio]` in any post or page where you want to display the Wordio form.

== Frequently Asked Questions ==

= Do I need a Twilio account to use Wordio? =

Yes. You must sign up for an account with Twilio to use Wordio. You also need a text enabled
device that texts will be sent to. Standard messaging rates apply.
